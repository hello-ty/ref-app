# 冷蔵庫アプリ

## 使用技術

- Nextjs
- Firebase(Firestore,Storage)
- Tailwindcss

## 要件定義

<p>冷蔵庫の中身を管理できる Web アプリを作りました。</p>
<p>買い物の最中に冷蔵庫の中身が確認できて、余分に買ってしまうのを防止できます。</p>

## 苦労した点

- Storage に画像をアップロードして、画面に表示させること

## 今後やりたいこと

- レシピ機能を追加する
- UI デザインを細部まで作り込む
